Uninstall Oracle on Windows

  1. Uninstall using Oracle universal installer
  2. Reboot
  3. Delete C:\Program Files\Oracle
  4. Delete ORACLE_HOME
  5. Delete ORACLE_DATA
  6. Repair registry using NU Windoc
  7. Delete registry key: HKLM\SYSTEM\CONTROLSET001\Enum\Root\LEGACY_ORACLE*
  8. Delete registry key: HKLM\SOFTWARE\ORACLE
  9. Reboot

Migrating Oracle on Windows

  • Copy files
    • $ORACLE_HOME/dbs init$SID.ora
    • $ORACLE_HOME/admin Directory
    • $ORACLE_HOME/network/admin all files
  • Create new SID instance
oradim -new -sid <SID> -pfile <full-init-ora-filename>

  • set env ORACLE_SID
  • update the listener.ora and tnsnames.ora. /Sample listener.ora, /Sample tnsnames.ora
  • Connect and startup
