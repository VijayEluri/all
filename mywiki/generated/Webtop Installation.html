Installation on Websphere Network Deploy Mode

  1. Install Websphere using root user
  2. set root’s DOCUMENTUM and DOCUMENTUM_SHARED
  3. run webtop installer
  4. install webtop into a location
  5. deploy webtop to cluster/server using deployment manager
  6. run webtop installer again
  7. deploy webtop to those server.
