Components

  • Library server
    • Need a database
  • Resource manager
    • Running inside a web app server
    • File can be stored distributely
    • Can be configured to do partial replication, providing failover
    • Need a database
  • CM Client for Windows
    • triangle: library server/resource manager/cm client
    • cm client *> library server: SQL request directly
    • library server *> cm client: token and locator
    • cm client *> resource manager: HTTP request
  • eClient (web based)
    • client capabilities: http://www-1.ibm.com/support/docview.wss?uid=swg21220064
  • Administration client
  • Information Integrator for Content (ECI like)
  • CM VideoCharger
  • CM OnDemand

vs DCTM

  1. No HP-UX support
  2. No Sybase, SQL Server support
  3. No lightweight cache component (BOCS)
  4. Need many database instances
  5. Doesn’t support D&D on web
  6. Cannot import multiple files through web
  7. Cannot manipulate compond document(virtual document) through desktop/web
  8. No web based administration client
